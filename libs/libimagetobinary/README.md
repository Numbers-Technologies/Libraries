## About
Convert image to binary format.
From ```path_to_file``` to ```path_to_save```.\
It can convert ```png```, ```jpg```, ```jpeg``` to ```txt```.

#### Dependencies
 - None


#### Methods:
```c
int convert_to_binary(char *path_to_file, char *path_to_save) -> int32 {...}
int main() -> int32 {...}
```
